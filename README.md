# Personal Finance Management App

A cross-platform mobile application that helps users **track daily expenses, categorize spending, and manage monthly budgets** with a simple and clean UI.

---

## Tech Stack

* **React Native (Expo)**
* **Firebase Firestore**
* **JavaScript**

---

##  Features

* Add and manage daily expenses
* Categorize spending (Food, Travel, Shopping, etc.)
* Set a monthly budget
* Automatic calculation of:

  * Total spent
  * Remaining balance
  * % Used
* Stores data in Firebase (real-time updates)
* Minimal and mobile-friendly interface

##  How to Run

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npx expo start
```

### 3. Run on device

* Install **Expo Go** app on phone
* Scan displayed QR code
